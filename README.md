# Assistente CMDPII

Assistente de IA para integração de alunos novatos no Colégio Militar Dom Pedro II.

## Protótipo navegável

```bash
cd wireframe
python serve.py
```

Abra **http://localhost:3456/index.html** (ou dê duplo clique em `wireframe/iniciar.bat` no Windows).

| Recurso | URL |
|---------|-----|
| Landing e fluxo | http://localhost:3456/index.html |
| Design system (vitrine) | http://localhost:3456/design-system/showcase.html |

Documentação: [`wireframe/README.md`](wireframe/README.md) · [`design-system/README.md`](design-system/README.md) · [`prd.md`](prd.md)
